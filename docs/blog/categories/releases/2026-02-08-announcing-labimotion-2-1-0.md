# 🎉 LabIMotion 2.1.0 Released

{% include post-header.html
  description=page.description
  author=page.author
  date=page.date
%}

We are pleased to announce **LabIMotion 2.1.0**, a release focused on improving workflow efficiency and data management.

Key updates include:

- **Dynamic Layer Display Names** – Automatically reflect data in layer titles for easier identification
- **Layer Grouping** – Manage multiple layers as a single unit with ease
- **Enhanced Generic Datasets** – Add source references, extended context, and associate elements directly within datasets
- **Validation & Restriction Improvements** – Smarter checks and easier configuration
- **Sample Linking in Generic Segments** – Directly associate samples with segments
- **Export Table Data to XLSX** – Convenient offline access to table data
- **User Interface Improvements** – Resizable workspace and layout refinements for a smoother experience

This release also includes bug fixes and internal improvements for better stability and maintainability. A complete list of changes is available in the [Release 2.1.0](https://github.com/LabIMotion/labimotion/blob/main/CHANGELOG.md#210){:target="_blank" rel="noopener"}.
